Python 3.7.3

Основные задания + дополнительные
=================================

1 задание - сортировка слияниями.
Для сртировки чисел необходимо вызвать метод sort(),в который передаются название файла с числами,
название файла, в который будет записан результат сортировки, и количество чисел в одном файле в нчале сортировки
(по умолчанию 100000)

2 задание - преобразователь в json/
На входе - объект python, на выходе - строка json.

3 задание класс n-мерный вектор.
В задании создан класс Vector, в котором перегружены операции для работы с ним: +, =, *, ==, != ...

4 задание - декоратор.
Если функция уже вызывалась с такими аргументами, то ф-ия вернет результат, иначе посчитает его и вернут

5 задание - тесты
Необходимо запустить файл с тестами. Тесты написаны под все задания. Время теста ~ 20
